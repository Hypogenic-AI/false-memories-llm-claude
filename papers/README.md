# Downloaded Papers

## Core Papers (Directly on Topic)

1. **Why do LLMs freak out over the seahorse emoji?** (Vogel, 2026)
   - Source: Blog post (vgel.me/posts/seahorse/) — not a PDF; technical analysis captured in literature_review.md
   - Why relevant: Primary mechanistic investigation of the seahorse emoji hallucination using logit lens

2. **Analyzing Memory Effects in LLMs through Cognitive Psychology** (Cao et al., 2025)
   - File: `cao2025_memory_effects_llm_cognitive_psychology.pdf`
   - arXiv: 2509.17138
   - Why relevant: DRM false memory paradigm applied to LLMs; 11.4% critical lure recall rate

3. **When Agents "Misremember" Collectively: Mandela Effect in LLM Multi-Agent Systems** (Xu et al., ICLR 2026)
   - File: `xu2026_mandela_effect_multiagent.pdf`
   - arXiv: 2602.00428
   - Why relevant: Collective false memory benchmark (MANBENCH); mitigation strategies

4. **Conversational AI Amplifies False Memories in Witness Interviews** (Chan et al., 2024)
   - File: `chan2024_conversational_ai_false_memories.pdf`
   - arXiv: 2408.04681
   - Why relevant: LLMs induce 3x false memories in humans; downstream impact of confabulation
